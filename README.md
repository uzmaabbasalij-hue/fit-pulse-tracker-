# Fit Pulse 🏋️‍♂️💪

Fit Pulse is a responsive and user-friendly fitness tracking web application designed to help fitness enthusiasts log their daily workout routines, track sets/reps, and maintain a consistent exercise schedule.

## ✨ Features
- **Workout Logger:** Easily create and log daily workout routines (Cardio, Strength, Flexibility).
- **Custom Exercises:** Add specific exercises with detailed sets, repetitions, and weights.
- **Streak Tracker:** Keep a visual count of consecutive days you have completed a workout.
- **Timer Integration:** Built-in stopwatch and countdown timer for resting between sets.

## 🛠️ Tech Stack
- HTML5 / CSS3 (Responsive Design)
- JavaScript (ES6+ / LocalStorage for data saving)
